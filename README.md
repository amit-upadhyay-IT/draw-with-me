> socket based app for white board sharing

## Installation and Usage

Step 1:
```sh
$ git clone https://github.com/amit-upadhyay-IT/draw-with-me.git
```
Step 2:
```sh
$ cd draw-with-me
```
Step 3:
```sh
$ npm i
```
Step 4:
```sh
$ node index.js
```

Step 5:
visit : [http://localhost:3000](http://localhost:3000) and share the url with others.
